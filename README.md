# SemanaOmniStack11
Be the Hero feito na semana OmniStack 11 da  Rocketseat usando **NodeJS**, **React**, **React Native**, **Express**, **SQLite**, **HTML5**, **CSS3**, **Celebrate**, **Jest**, **Knex**

------------


![Tela do frontend](https://i.imgur.com/Z2wva6A.png "Tela do frontend")
## Instalação
### Node - Backend
`npm install`
### React  - Frontend
`npm install`
### React Native - Mobile
`yarn install`

------------


## Iniciando
### Node - Backend
 `npm start`
 ### React  - Frontend
`npm start`
### React Native - Mobile
`yarn start`

------------


 ## Teste
 ### Node - Backend
 `npm test`

------------
